# Lancashire Light

A single-page web app to quickly check sunrise and sunset times across Lancashire.

Live site: [mrjohnholt.github.io/sunriseSunset](https://mrjohnholt.github.io/sunriseSunset/)

## About

Get  sunrise and sunset times for any date. Built as a clean, no-fuss utility with no login or tracking required.

## Tech

- Plain HTML, CSS, and JavaScript
- Hosted on GitHub Pages
